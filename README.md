# Curso de JavaScript - Fundamentos e Prática 📚

Este repositório é dedicado ao estudo da linguagem **JavaScript**, contendo exercícios, exemplos de lógica de programação e pequenos projetos desenvolvidos durante o meu processo de aprendizagem. 

O objetivo aqui é consolidar os conceitos fundamentais que servem de base para a automação de testes com frameworks modernos (Cypress, Playwright, etc.).

## 🚀 Conceitos Explorados

O projeto abrange tópicos essenciais da linguagem:

- **Básico:** Variáveis (`let`, `const`), tipos de dados, operadores aritméticos e lógicos.
- **Estruturas de Controle:** Condicionais (`if/else`, `switch`) e laços de repetição (`for`, `while`, `forEach`).
- **Funções:** Declarações, expressões, Arrow Functions e funções de Callback.
- **Manipulação de Dados:** Métodos de Array (`map`, `filter`, `reduce`), manipulação de Objetos e JSON.
- **Assincronismo:** Promises e `async/await` (crucial para testes E2E).
- **DOM:** Manipulação de elementos HTML via JavaScript (se aplicável ao curso).

## 🛠️ Tecnologias

* [JavaScript (ES6+)](https://developer.mozilla.org/pt-BR/docs/Web/JavaScript) - Linguagem base.
* [Node.js](https://nodejs.org/) - Para execução dos scripts via terminal.
